# mmo-server
## Simulates a slice of universe!

```
USAGE:
    mmo-server [OPTIONS]

FLAGS:
    -h, --help       Prints help information
    -V, --version    Prints version information

OPTIONS:
    -a, --address <addr>      Host to connect to address:port
    -e, --expiration <exp>    Connection expiration limit
    
    
    
INSTALATION:
Install https://www.rustup.rs
git clone git@github.com:RedPillVR/mmo.git
cd mmo/mmo-server
cargo build

RUN:
cargo run -- -a IP_OF_SERVER
```
